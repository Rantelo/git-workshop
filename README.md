## Cheat Sheet

* git clone
* git status
* git add <path/to/file>
* git commit -m “Explain shortly what you did here”
* git branch
* git fetch
* git checkout <branch_name>
* git checkout -b <new_branch>
* git push origin <branch_name>
* git pull origin <branch_name>
* git merge <branch_name>
* git log
